# ⛽ Fuel Price Data Analysis Project

## 📌 Overview
This project focuses on analyzing fuel price trends using a dataset containing petrol, diesel, LPG prices, crude oil prices, tax percentages, and country information. The goal is to extract meaningful insights using data cleaning, preprocessing, and visualization techniques.

---

## 📊 Objectives
- Perform data cleaning and preprocessing  
- Handle missing values and incorrect data types  
- Encode categorical variables  
- Analyze relationships using correlation  
- Visualize trends and patterns  
- Draw meaningful conclusions  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Seaborn  
- Matplotlib  

---

## 🧹 Data Preprocessing
- Checked and handled missing values using `dropna()`  
- Converted data types (e.g., date to datetime format)  
- Encoded categorical column (`country`) using one-hot encoding  
- Removed unnecessary or redundant columns  

---

## 📈 Exploratory Data Analysis (EDA)

### 🔹 Correlation Analysis
- Generated correlation matrix to identify relationships between variables  
- Visualized using heatmap  

### 🔹 Visualizations
- 📊 Bar Plot → Country vs Petrol Price  
- 📉 Histogram → Distribution of Petrol Prices  
- 📈 Line Plot → Fuel Price Trends Over Time  
- 🔗 Scatter Plot → Crude Oil vs Petrol Price  

---

## 🔍 Key Insights
- Petrol, diesel, and LPG prices are highly correlated  
- Crude oil prices have a moderate impact on fuel prices  
- Tax percentage significantly influences fuel prices  
- Fuel prices show trends over time  
- Country-wise comparison provides regional insights  

---

## ⚠️ Observations
- Strong multicollinearity found among fuel types  
- Encoded categorical variables should be used carefully for correlation  
- External factors like policies affect fuel pricing  

---

## 🚀 Conclusion
The analysis shows that fuel prices are primarily driven by crude oil trends and taxation. Strong relationships exist among different fuel types, indicating similar pricing behavior. Visualization and correlation techniques helped uncover patterns and dependencies within the dataset.
